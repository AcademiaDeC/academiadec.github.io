Lenguaje C — Academia Suprema

Historia de C

El lenguaje C fue creado en 1972 por Dennis Ritchie en Bell Labs. Fue diseñado para implementar UNIX. Su objetivo era combinar eficiencia de bajo nivel con sintaxis estructurada. C influyó en casi todos los lenguajes modernos.

¿Qué es C?

C es un lenguaje compilado, estructurado y de propósito general. No tiene recolector de basura. El programador controla directamente la memoria. Es rápido porque se compila a código máquina.

Proceso de Compilación

gcc programa.c -o programa

Fases: 1. Preprocesador 2. Compilador 3. Ensamblador 4. Linker

Recursión

int factorial(int n){
    if(n==0) return 1;
    return n * factorial(n-1);
}

Operadores Bitwise

int a = 5;  // 0101
int b = 3;  // 0011

int and = a & b;  // 0001
int or = a | b;   // 0111
int xor = a ^ b;  // 0110

Proyecto Final — Sistema de Gestión

#include <stdio.h>
#include <stdlib.h>

struct Estudiante{
    char nombre[50];
    float nota;
};

int main(){
    struct Estudiante lista[3];

    for(int i=0;i<3;i++){
        printf("Nombre: ");
        scanf("%s", lista[i].nombre);

        printf("Nota: ");
        scanf("%f", &lista[i].nota);
    }

    printf("\nLista:\n");
    for(int i=0;i<3;i++){
        printf("%s - %.2f\n", lista[i].nombre, lista[i].nota);
    }

    return 0;
}

Este proyecto combina: - Arrays - Structs - Entrada de datos - Salida formateada
