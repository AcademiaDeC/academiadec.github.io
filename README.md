# C Academia Suprema

C Academia Suprema es una plataforma web educativa diseñada para enseñar **programación en C** desde lo más básico hasta conceptos avanzados. Cada sección incluye explicaciones teóricas y ejemplos prácticos de código.

## Características

- Historia y origen del lenguaje C.
- Tipos de datos, variables y operadores.
- Control de flujo: if, else, for, while.
- Funciones y paso de parámetros.
- Arrays y punteros.
- Memoria: stack y heap.
- Structs y organización de datos.
- Manejo de archivos.
- Algoritmos y estructuras de datos.
- Recursión y operadores bitwise.
- Proyecto final de ejemplo.
- Código bien formateado y explicado.
- Sidebar de navegación para acceder rápidamente a cada sección.

## Tecnologías

- HTML5
- CSS3
- JavaScript (para funcionalidades opcionales como copiar código)

## Uso

1. Clona o descarga este repositorio.
2. Abre `c-academia-suprema.html` en tu navegador.
3. Navega por las secciones desde el sidebar.
4. Copia el código de ejemplo y practícalo en tu compilador local de C (GCC, Clang, Dev-C++, etc.).

## Nota

Este proyecto es **educativo** y funciona como **manual interactivo de C**.  
No incluye compilador real en línea, pero puedes copiar el código a tu entorno de desarrollo local para ejecutarlo.

¡Aprende C paso a paso y domina la programación a bajo nivel!
